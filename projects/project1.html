Self Opening Bin

The idea of a self opening bin came from the need to create a more efficient waste disposal system.

Back to Portfolio

Details

This project was built using Arduino for the sensor and motor control, with a focus on creating a user-friendly interface.

I created the prototype using Tinkercad, which allowed me to simulate the circuit and test the design before building the physical model.
